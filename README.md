# Lengua Extranjera IV - Website

Proyecto web para la materia Lengua Extranjera IV de la Universidad de Cundinamarca - Extensión Soacha.

## Estructura del Proyecto

```
EnglishPage/
├── html/                  # Archivos HTML
│   ├── index.html        # Página principal
│   ├── folleto.html      # Carrusel de imágenes "Finland Brochure"
│   └── evolucion.html    # Página de evolución
├── css/                   # Archivos de estilos
│   └── styles.css        # Estilos principales
├── js/                    # Archivos JavaScript
│   ├── script.js         # Funcionalidades generales
│   └── carousel.js       # Funcionalidades del carrusel
├── assets/                # Recursos estáticos
│   ├── images/           # Imágenes y recursos visuales
│   └── fonts/            # Fuentes tipográficas
└── README.md             # Documentación del proyecto
```

## Características

- **Diseño Responsive**: Adaptable a dispositivos móviles, tablets y desktop
- **Navegación Intuitiva**: Menú principal con botones de navegación
- **Carrusel de Imágenes**: Galería interactiva con 6 imágenes (317x687px)
- **Animaciones y Transiciones**: Efectos visuales modernos
- **Navegación Múltiple**: Botones, teclado, touch/swipe support

## Tecnologías Utilizadas

- **HTML5**: Estructura semántica
- **CSS3**: Estilos modernos con gradientes y animaciones
- **JavaScript ES6+**: Funcionalidades interactivas
- **Responsive Design**: Media queries para diferentes dispositivos

## Uso

1. Abrir `html/index.html` en un navegador web
2. Navegar usando los botones principales
3. Explorar el carrusel en la sección "Folleto"

## Desarrollo

El proyecto está organizado modularmente con separación clara de responsabilidades:
- **HTML/**: Contenido y estructura
- **CSS/**: Presentación y estilos  
- **JS/**: Comportamiento e interactividad
- **assets/**: Recursos estáticos
